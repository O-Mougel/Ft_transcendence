ft_transcendence
Description

ft_transcendence est un projet de développement web consistant à créer une plateforme permettant de jouer au jeu Pong directement dans un navigateur.
L'application propose un système de jeu multijoueur en temps réel, avec une interface web interactive et des fonctionnalités sociales autour du jeu.

Le projet inclut également un système de tournoi, permettant à plusieurs joueurs de s’affronter et de suivre les matchs et les résultats.

Fonctionnalités principales

Jeu Pong jouable directement sur le site

Multijoueur en temps réel

Système de tournoi avec matchmaking

Interface web interactive

Gestion des joueurs et des alias

Sécurité des données et des entrées utilisateurs

Ces fonctionnalités constituent la base obligatoire du projet avant l’ajout de modules avancés.

Technologies

Le projet repose sur une architecture web moderne :

Frontend : TypeScript (SPA)

Backend : PHP ou framework selon les modules choisis

Conteneurisation : Docker

Navigateur cible : Mozilla Firefox (dernière version stable)

Objectif du projet

Ce projet vise à développer la capacité à concevoir une application complexe avec des technologies parfois nouvelles, en mettant l’accent sur la conception, la collaboration et la résolution de problèmes.

Lancer le projet
```bash
> make
```

L’application sera ensuite accessible depuis votre navigateur.
